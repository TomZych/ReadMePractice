# HashMap Class Implementation
This project showcases a standard C++ implementation of the HashMap class.

## Functions
- **insert** - Inserts a value based off key hash mapping. Resizes on load factor break.
- **at** - Returns reference to value given a key and taking its hash
- **contains** - Returns true or false based off keys precense in map
- **erase** - deletes mapping for given key, makes sure no chain is broken.



## Constructor Utilized
- **Default** 
- **Copy**
- **Assignment**
- **Destructor**
